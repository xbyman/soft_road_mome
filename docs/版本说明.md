# 版本说明

## 1. 版本信息

| 项目 | 内容 |
| --- | --- |
| 软件名称 | `road_mome_demo_v1` |
| 版本号 | V1.0 |
| 版本定位 | Road-MoME 单机浏览器演示平台 |
| 发布形态 | 本地离线演示版本 |

## 2. 本版本主要完成内容

V1.0 当前已完成以下内容：

- 完成单机浏览器页面，可在本地启动并访问。
- 完成启动前校验，用于检查配置、权重、演示数据和输出目录。
- 完成样本运行前校验，用于拦截不满足要求的异常样本。
- 完成基于预置样本的单帧检测流程。
- 完成四宫格结果显示。
- 完成图像、JSON 和 CSV 导出功能。
- 完成异常路径控制，避免因单个样本异常导致页面直接崩溃。
- 完成本地运行日志记录。
- 完成项目结构拆分与模块化整理，形成较清晰的目录与职责边界。

## 3. 本版本未包含内容

V1.0 当前未包含以下能力：

- 真实硬件接入
- 实时流处理
- 批量处理
- 在线训练
- 在线学习更新
- 车载控制接口联动

## 4. 已知边界

当前版本的主要边界如下：

- 本版本是单机离线演示平台，不是商用系统。
- 本版本面向预置样本演示，不面向实时传感器输入。
- 本版本支持单帧运行与导出，不支持连续流式处理。
- 本版本强调“正常路径可用、异常路径可控、页面不崩”，不代表已完成生产化验证。
- 本版本不包含性能压测、并发能力和车载部署能力验证。

## 5. 后续扩展建议

后续可在当前 V1.0 基础上考虑以下扩展方向：

- 增加真实数据源接入能力，如相机、雷达或其他采集接口。
- 从单帧演示扩展到连续帧或实时流处理。
- 增加批量样本处理与批量导出能力。
- 完善结果管理、日志管理和配置管理能力。
- 视需要扩展为服务化部署或更完整的前后端分离架构。
- 在后续研究版本中增加训练、更新或联动控制相关能力。

## 6. 说明

V1.0 的主要价值在于形成一个结构清晰、可运行、可演示、可说明的本地演示平台，用于课程设计展示、答辩说明和后续版本迭代的基础。当前版本已具备基本功能闭环，但仍属于演示性质版本。
